Static Semantics

PropName

PropName returns ~empty~ for hidden method definitions

 PropertyDefinition : HiddenMethod ClassElement : HiddenMethod 1. Return ~empty~.

HiddenMethodName

HiddenMethodName of a |HiddenMethod| is ~empty~ unless it derives a method with an |IdentifierName| property name.

 HiddenMethod : `hidden` MethodDefinition 1. Return HiddenMethodName of |MethodDefinition|. HiddenNameList

HiddenNameList of a |PropertyDefinitionList| or |ClassElementList| is a List containing all of the hidden names defined in an object literal or class body. PropertyDefinitionList : PropertyDefinition 1. Return HiddenNameList of |PropertyDefinition|. PropertyDefinitionList : PropertyDefinitionList `,` PropertyDefinition 1. Let _names_ be HiddenNameList of |PropertyDefinitionList|. 2. Append to _names_ the elements of the HiddenNameList of |PropertyDefinition|. 3. Return _names_. PropertyDefinition : IdentifierReference PropertyDefinition : PropertyName `:` AssignmentExpression PropertyDefinition : MethodDefinition 1. Return a new empty List. PropertyDefinition : HiddenMethod 1. Return a new List containing HiddenMethodName of |HiddenMethod|. ClassElementList : ClassElement 1. Return HiddenNameList of |PropertyDefinition|. ClassElementList : ClassElementList ClassElement 1. Let _names_ be HiddenNameList of |ClassElementList|. 2. Append to _names_ the elements of the HiddenNameList of |ClassElement|. 3. Return _names_. ClassElement : MethodDefinition ClassElement : `static` MethodDefinition ClassElement : `;` 1. Return a new empty List. ClassElement : HiddenMethod ClassElement : `static` HiddenMethod 1. Return a new List containing HiddenMethodName of |HiddenMethod|. InstanceVariableList : BindingIdentifier 1. Return a new List containing StringValue of |BindingIdentifier|. InstanceVariableList : InstanceVariableList `,` BindingIdentifier 1. Let _names_ be HiddenNameList of |InstanceVariableList|. 2. Append StringValue of |BindingIdentifier| to _names_. 3. Return _names_.
